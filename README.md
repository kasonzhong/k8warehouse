# K8 退货处理系统 V10（单元格图片＋腾讯文档复制版）


V9新增退货单跨账户共享，并保留V8的独立账户、云端照片和电脑/手机同步。

## 重要：先修复当前数据库错误

如果网页显示：

```text
Could not find the table 'public.return_batches' in the schema cache
```

说明数据库初始化SQL尚未成功运行。请在Supabase：

```text
SQL Editor → New query
```

复制并运行：

```text
supabase-setup-v9.sql
```

V9 SQL会一次性创建V8所需表以及新的共享权限表。

## 新增共享功能

退货单创建者可点击卡片上的“共享权限”：

1. 输入对方已经注册的邮箱
2. 选择“仅查看”或“可以编辑”
3. 发送申请
4. 对方在“共享邀请”中接受
5. 退货单出现在对方工作台

### 仅查看

对方可以：
- 查看退货单
- 查看和放大照片
- 导出Excel

不能：
- 修改SKU、面单号、照片或质检
- 删除任务
- 确认完成
- 继续分享或删除整张退货单

### 可以编辑

对方可以修改退货内容和上传/删除照片，但只有创建者可以：
- 继续共享权限
- 删除整张退货单

## 删除按钮

每张自己创建的退货单卡片新增“删除退货单”。删除时会同时删除：
- 退货单数据
- 该退货单的照片
- 所有共享邀请和共享权限

## 更新步骤

1. 运行 `supabase-setup-v9.sql`
2. 用V9文件替换GitHub仓库内容
3. 不要覆盖你已经配置好的 `supabase-config.js`
4. GitHub Desktop Commit并Push
5. 等待Vercel显示Ready
6. 使用两个不同邮箱测试共享


## V10 导出更新

### Excel 单元格图片

默认 Excel 导出改为使用 Microsoft Excel 的 `IMAGE` 单元格公式：

- 图片属于单元格内容，不再是悬浮在单元格上方的绘图对象
- 调整行高、列宽时图片会跟随单元格
- 排序和筛选时图片会跟随所在行
- 点击图片单元格可以打开云端原图
- “高清照片”工作表仍然保留

兼容要求：

- Microsoft 365
- Excel 2024
- 支持 IMAGE 函数的新版 Excel

旧版 Excel、部分 WPS 和腾讯文档可能显示公式不兼容。

图片链接使用 Supabase 私有存储生成的 365 天签名地址。任何获得导出文件的人，在有效期内都能通过链接查看对应照片。

### 腾讯文档复制

每张退货单新增“复制到腾讯文档”：

1. 网站直接读取退货图片
2. 生成带图片的 HTML 表格
3. 尝试写入浏览器富文本剪贴板
4. 在腾讯文档表格中选择 A1 后粘贴
5. 同时下载一个自包含的 HTML 备用文件

如果直接粘贴只出现文字：

1. 打开下载的 `腾讯文档复制版.html`
2. 等待图片显示
3. Command+A
4. Command+C
5. 在腾讯文档 A1 粘贴

真正稳定的“一键创建腾讯共享表格并插入单元格图片”需要申请腾讯文档开放平台应用，并通过 OAuth 和后端调用图片上传、单元格图片接口。
