K8 CLOUD SETUP

需要连接 Supabase

网站代码已经包含账户和云端同步功能，但还需要填写你自己的 Supabase 项目地址和 Publishable Key。

配置方法
  1. 在 Supabase 创建项目。
  2. 运行项目内的 supabase-setup.sql。
  3. 打开 supabase-config.js，填入 Project URL 和 Publishable Key。
  4. 提交到 GitHub，等待 Vercel 自动重新部署。

请勿填写或公开 Service Role Key。浏览器只应使用 Publishable Key 或旧版 anon key。
